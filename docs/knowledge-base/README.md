# 知识库

这里用于记录项目开发过程中遇到的问题、排查过程和最终解决方案。

建议按主题拆分文档，例如：

- `pptx-package.md`：PPTX 包结构、关系解析、路径处理相关问题。
- `xml-parser.md`：XML 解析、命名空间、节点查询相关问题。
- `renderer-svg.md`：SVG 渲染、坐标、样式、图片显示相关问题。
- `text-layout.md`：文本排版、字体、换行、缩进相关问题。
- `diagnostics.md`：诊断错误码、降级策略、异常样本相关问题。

每条记录建议包含：

```md
## 问题标题

### 现象

### 原因

### 解决方案

### 关联文件
```
